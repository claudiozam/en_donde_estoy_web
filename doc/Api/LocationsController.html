Locations

Obtiene dispositivos cercanos. El mismo permite realizar filtrado por categoria del dispositivo. #

/api/locations/find_near_locations/:latitude/:longitude/:category_name [GET]

Required Parameters

  • latitude integer Latitud del dispositivo que solicita posiciones de dispositivos cercanas. Este valor es recibido por url
  • longitud integer Longitud del dispositivo que solicita posiciones de dispositivos cercanas. Este valor es recibido por url
  • category_name string Nombre de la categoria a la que pertenece el dispositivo buscado. (En caso de que no se quiera filtrar por este campo se lo envia como "all"). Este valor es recibido por url.

Response Fields

  • message string String con el resultado del metodo.
  • list array Array con los location_points encontrados. Dentro del array habran arrays con informaci√≥n de cada location encontrado.
    • latitude Valor de la latitud. Actualmente es un integer sin formato
    • longitude Valor de la longitud. Actualmente es un integer sin formato
    • updated_at Fecha de creacion del mismo. El formato del mismo es "%Y-%m-%d %H:%M" (Eg: 2013-03-24 18:06:29)
    • category Nombre de la categoria a la que pertenece.
    • device Nombre el dispositivo.
    • device_description Texto con la descripcion del dispositivo.
  • code string numero que representa el resultado del request.
    • 000 SUCCESS
    • 600 No se encontraron location points.
    • 601 No se pasaron longitud o latitud en la url.
    • 999 ERROR

Examples

Request $.ajax({ type: 'GET', url: "/api/locations/find_near_locations/30.5/21.5/all" }) Response {"code":"000","list":[{"latitude":25.158,"longitude":30.588,"updated_at":"2013-04-14 19:18","category":"taxi","device":"Taxi_157","device_description":"taxi con id 157"},{"latitude":25.5,"longitude":35.0,"updated_at":"2013-04-14 20:08","category":"personal","device":"XuanJuan","device_description":"Abierto de 9hs a 21hs"}]}

Obtiene el ultimo location_point del dispositivo que se desea buscar.

/api/locations/:device_name/get_location [GET]

Required Parameters

  • device_name string Nombre del dispositivo buscado.

Response Fields

  • location_point Hash Un hash con los datos del ultimo location_point del dispositivo.
    • latitude Valor de la longitud.
    • longitude Valor de la latitude.
    • created_at Fecha en que fue actualizada la location. El formato del mismo es "%Y-%m-%d %H:%M" (Eg: 2013-03-24 18:06:29)
  • message string String con el resultado del metodo.
  • code string numero que representa el resultado del request.
    • 000 SUCCESS
    • 405 No se encontro el device solicitado.
    • 601 No se paso nombre del device o el mismo es "blanco".
    • 999 ERROR

Examples

Request $.ajax({ type: 'GET', url: "/api/locations/XuanJuan/get_location" }) Response "{"location_point":[{"latitude":25.5,"longitude":35.0,"created_at":"2013-04-14 20:08"}],"code":"000"}"

Obtiene toda las categorias posibles para locations.

/api/locations/get_all_categories [GET]

Response Fields

  • categories array Lista de hashes con los datos de cada categoria.
    • id Id de la categoria (valor integer).
    • nombre Nombre de la categoria.
    • descripcion Descripcion de la categoria.
  • code string numero que representa el resultado del request.
    • 000 SUCCESS
    • 999 ERROR

Examples

Request $.ajax({ type: 'GET', url: "/api/locations/get_all_categories" }) Response "{"categories":[{"id":1,"name":"personal","description":null},{"id":2,"name":"comercio","description":null},{"id":3,"name":"instituto","description":null},{"id":4,"name":"taxi","description":null},{"id":5,"name":"colectivo","description":null}],"code":"000"}"

Obtiene todos los tipos posibles para locations.

/api/locations/get_all_types [GET]

Response Fields

  • types array Lista de hashes con los datos de cada categoria.
    • id Id del tipo de location (valor integer).
    • nombre Nombre del tipo de location.
    • dinamic Valor entero que representa si la location es dinamica. (1 = verdadero, 0 = falso)
    • descripcion Descripcion del tipo de location.
  • code string numero que representa el resultado del request.
    • 000 SUCCESS
    • 999 ERROR

Examples

Request $.ajax({ type: 'GET', url: "/api/locations/get_all_types" }) Response "{"types":[{"id":1,"name":"establecimiento","dinamic":false,"description":null},{"id":2,"name":"movil","dinamic":false,"description":null},{"id":3,"name":"transporte","dinamic":true,"description":null}],"code":"000"}"
