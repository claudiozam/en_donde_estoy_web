<h2> Some examples with curl </h2>

Devices: 
<pre>
curl -v -k -X POST -d "name=SebasDeviceHey&category_id=1&type_id=1" http://127.0.0.1:3000/api/devices/create

{"message":"Device was created successfuly.","code":"600"}
</pre>

<pre>
curl -v -k -X PUT -d "latitude=20&longitude=25" http://127.0.0.1:3000/api/devices/1/update_location

{"message":"Updated sucessfully.","code":"000"}
</pre>
